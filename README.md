SnapLink - URL Shortner

SnapLink is a simple app that shrinks long URLs into shortlinks which you can share.

● Project Structure
  ⮡ backend/ → Node.js/Express server does the URL shrinking.
  ⮡ frontend/ → React frontend where users can give there long URLs and get the shirtend links.

● Features
 → Shorten any URL easily.

● How to Run
➜ Clone the repository
    git clone https://github.com/01-NiTiN/SnapLink-URL-Shortener.git
    cd SnapLink-URL-Shortener

➜ Install Dependencies
    cd backend
    npm install
    cd ../frontend
    npm install

➜ Return to root and start both frontend and backend
    cd ..
    npm start
